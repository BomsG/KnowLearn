# KnowLearn

**KnowLearn** is an educational web application designed to help students and teachers create, share, and complete quizzes, surveys, and learning assessments. The platform makes learning interactive and engaging, helping users test knowledge and track progress efficiently.

## Features

- **Quiz & Assessment Creation:** Teachers can create quizzes or learning assessments with multiple question types (multiple choice, true/false, short answer).
- **Student Interaction:** Students can take quizzes online and get instant feedback.
- **Progress Tracking:** Track student performance over time and analyze results.
- **Shareable Links:** Easily share quizzes with students via a link or email.
- **Responsive Design:** Works on desktop, tablet, and mobile devices.
- **Data Export:** Export results for offline review or grading.
